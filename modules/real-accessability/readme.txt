== Description ==

Real Accessability plugin adds very easy to use & intuitive accessability toolbar to your website.

**Real Accessability Features:**

* Increase font/text size
* Decrease font/text size
* Turn Black&White view
* Invert website colors
* Highlight all links with underline
* Use more readable font (Arial)
* Remove target attribute
* Add role="link" to links
* RTL & LTR Support
